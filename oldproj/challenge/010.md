# 010. 삼각형

입력받은 세 자연수를 삼각형의 각각의 변의 길이로 할 때, 직각삼각형인지 판별하시오.   
직각삼각형이면 첫째 줄에 'True' 를, 아니라면 'False' 를 출력하고, 'False' 인 경우에는 둘째 줄에 무슨 삼각형인지 한글로 출력합니다.   
단, 직각삼각형에서 가장 긴 변은 빗변입니다.   
   
Input 1
```
10 24 26
```
Output 1
```
True
```
   
Input 2
```
3 7 11
```
Output 2
```
False
둔각삼각형
```
